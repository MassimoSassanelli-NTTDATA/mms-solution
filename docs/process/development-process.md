# Agentic Development Process

```text
discover-story
  -> refine-story
  -> create task issues
  -> attach tasks as sub-issues
  -> create story branch
  -> start-story
  -> start-task
  -> validate-story
  -> sync-story
  -> review
  -> merge
```

Agents must read `WORKSPACE.md` and `REPOSITORY_CONTEXT.md` when available.
